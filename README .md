🎮📚 Gaming vs Academic Performance Analysis (Excel Project)

📌 Overview


This project explores the relationship between gaming habits and academic performance using Excel. The dataset is analyzed to understand whether gaming time affects students’ grades, study habits, and overall academic outcomes.


🎯 Objective

Analyze impact of gaming hours on academic performance

Compare performance of heavy gamers vs light/no gamers

Identify patterns between study time and gaming time

Understand how lifestyle habits influence grades

📁 Dataset Information


The dataset includes student-related attributes such as:


Hours spent gaming

Study hours

Academic scores / grades

Sleep patterns (if included)

Attendance or performance indicators

Demographic details (if available)

📊 Key Insights

Students with moderate gaming time often perform better than extreme gamers

Excessive gaming may correlate with lower academic scores

Balanced routine (study + limited gaming) shows best results

Study hours have stronger positive impact than gaming has negative impact

Some students maintain high performance despite gaming habits

📈 Tools Used

Microsoft Excel

Pivot Tables

Charts & Graphs

Data Filtering & Sorting

Conditional Formatting

Basic Statistical Analysis

📌 Analysis Features

Gaming vs Academic Score comparison

Study Hours impact analysis

Performance grouping (High / Medium / Low)

Correlation-style visual insights

Student behavior segmentation
